Overview

What is Smart Lender?

Smart Lender is a Flask-based loan approval prediction application. Users enter applicant and financial details, and the app returns a machine learning prediction that helps estimate whether a loan may be approved.

Features

Implemented Features

  • Loan Approval Prediction: Automated estimation of loan eligibility based on user inputs.
  • Machine Learning Model: Powered by a trained XGBoost classification model.
  • Applicant Information: Interactive profile capturing age, dependents, education, and employment.
  • Financial Summary: Structured metrics showcasing incomes, requested loan amount, and terms.
  • Downloadable PDF Report: Professional 2-page banking report summary generation.
  • Rule-based Explanation: Clear, human-readable reasons explaining the prediction outcome.
  • Responsive Design: Clean, modern, grid-based dashboard for desktop and mobile viewports.

Selected Model

XGBoost

XGBoost was selected because it delivers the best balance of classification performance, stable predictions across training folds, and robust loan approval decision quality.
